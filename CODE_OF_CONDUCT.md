# Code of Conduct

## Our Pledge

We are committed to making this project a respectful, welcoming, and useful space for contributors, readers, and visitors.

Everyone interacting with this repository, whether as a contributor, issue reporter, or reviewer, is expected to treat others with professionalism and courtesy.

## Expected Behavior

Please:

- be respectful in all communication
- keep feedback constructive and specific
- focus on the code, content, or behavior rather than the person
- respect different experience levels and working styles
- help keep discussions clear, calm, and productive

## Unacceptable Behavior

The following behaviors are not acceptable:

- harassment or personal attacks
- abusive, hateful, or discriminatory language
- deliberate disruption of discussions or pull requests
- repeated unwanted messaging
- shaming contributors for asking questions or learning

## Project Culture

This repository is personal and portfolio-driven, but it is still open to contributions and discussion. The goal is to improve the project, not to create unnecessary conflict.

Keep feedback practical. If something should be changed, explain what should change and why.

## Enforcement

If behavior does not match this code of conduct, maintainers may:

- request a public or private apology
- ask for a comment or issue to be edited or removed
- reject a contribution
- block future participation if needed

## Scope

This code of conduct applies to:

- issues
- pull requests
- commit messages
- code review comments
- discussion in related project channels

## Attribution

This code of conduct is intended to keep the repository respectful and manageable. It is not a legal contract, but it is the behavioral standard for this project.
